# UI

## Muc dich
Chua tat ca man hinh va component UI cua MentorMe, duoc xay dung bang Jetpack Compose.

## Cau truc thu muc

- `navigation/` - AppNav, routes, deep links, bottom bar integration.
- `components/ui/` - shared UI atoms/molecules (MMButton, MMTextField, GlassDialog, glass helpers).
- `utils/` - helper formatting/utility functions used by UI.
- `layout/` - layout chung (bottom nav, header bar).
- `auth/` - man hinh dang nhap/dang ky.
- `booking/` - booking flow va chi tiet booking.
- `calendar/` - lich va availability.
- `chat/` - chat screens.
- `chat/components/` - chat UI pieces (composer, bubbles, conversation cards).
- `dashboard/` - mentor dashboard.
- `home/` - trang chu.
- `mentors/` - danh sach mentor va chi tiet.
- `notifications/` - thong bao, in-app + push.
- `onboarding/` - onboarding flow.
- `profile/` - profile entry screens.
- `profile/components/` - profile tabs, fields, and settings sections.
- `review/` - review UI.
- `search/` - search mentor + filter UI.
- `session/` - session flow.
- `theme/` - theme, colors, liquid glass.
- `videocall/` - video call UI.
- `wallet/` - vi, topup, withdraw.
